# 📚 Projeto Escolar - Aplicação React

Este é um projeto escolar desenvolvido usando **React.js**, com o objetivo de aprender e aplicar os conceitos básicos de desenvolvimento de aplicações web modernas com componentes reutilizáveis, estado e rotas.

## 🧠 Objetivo

O objetivo deste projeto é criar uma aplicação front-end interativa, responsiva e funcional, como parte de uma atividade escolar para a disciplina de [Nome da Disciplina].

## 🚀 Funcionalidades

- ✅ Interface amigável e responsiva
- ✅ Componentes reutilizáveis
- ✅ Navegação entre páginas com React Router
- ✅ Gerenciamento de estado com useState/useEffect
- ✅ Integração com API (se aplicável)
